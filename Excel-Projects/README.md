# Sleep Disorders Analysis (Excel)

## Overview
This project analyzes factors influencing sleep disorders using Excel. It includes data cleaning, descriptive statistics, and visualization.

## Tools Used
- Microsoft Excel (Pivot tables, charts, formulas)

## Steps Taken
- Cleaned and structured the dataset
- Created pivot tables and charts
- Analyzed patterns across age, gender, stress, and sleep duration
- Highlighted key insights and trends

## Key Insights
- Females reported more sleep disorders than males
- Higher stress levels correlated with shorter sleep duration
- Age group 30–40 showed the highest prevalence of sleep disorders
