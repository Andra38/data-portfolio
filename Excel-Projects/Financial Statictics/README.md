# 📊 Financial Dashboard Suite in Microsoft Excel

An interactive financial analytics dashboard built entirely in *Microsoft Excel*, showcasing advanced dashboard design, data visualization, KPI reporting, and business intelligence techniques without using Power BI or Tableau.

This project demonstrates how Excel can be transformed into a powerful business reporting tool using Pivot Tables, Pivot Charts, Slicers, Form Controls, Named Ranges, Conditional Formatting, and dynamic formulas.

---

## 📸 Dashboard Preview

### 1. Income Source Dashboard

![Income Source Dashboard](income-source.jpg)

Provides an overview of revenue generated from different income streams including:

- Total Financial Statistics
- Income Target vs Actual Income
- Income Achievement Gauge
- Revenue Breakdown by Source
- Monthly Operating Cost
- Monthly Average Income
- Product, Subscription, Consulting, Affiliate and Online Income Analysis

---

### 2. Geography Dashboard

![Geography Dashboard](geography.jpg)

Visualizes business performance across multiple countries.

Key insights include:

- Revenue by Country
- Geographic Revenue Distribution
- Sales Achievement %
- Total Cost vs Total Profit
- Profit Ratio
- Interactive Year Filter
- Country Performance Comparison

---

### 3. Sales Process Dashboard

![Sales Process Dashboard](images/salesprocess.jpg)

Analyzes sales pipeline performance from lead generation to closed deals.

Features include:

- Top Performing Month
- Best Sales Representative
- Conversion Rate
- Average Deal Size
- Total Deals Closed
- Sales Funnel
- Monthly Sales Trend
- Closed Deals by Sales Rep
- Sales Conversion Gauge
